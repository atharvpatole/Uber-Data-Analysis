# New York City Uber Trends

Analysis of Uber pickups in New York City from April to September 2014. This project explores trip patterns by hour, day, weekday, and month using data visualization and statistical analysis.

## Overview

- **Dataset**: Uber raw data (Apr-Sep 2014)
- **Analysis**: Trip patterns, peak hours, weekday vs weekend trends, monthly growth
- **Tools**: Python, Pandas, Matplotlib, Seaborn
